# Security Policy

## Supported Versions
本リポジトリでは `main` ブランチのみをサポート対象とします。

## Reporting a Vulnerability
脆弱性やセキュリティ上の懸念を見つけた場合は、**Issue を立てずに** 以下の方法でご連絡ください。

- GitHub の Private Vulnerability Reporting

## Notes
- P2P 技術を利用するため、URL 共有や画面・音声の公開範囲には十分注意してください
- ログや共有内容に機微情報を含めないでください
- 開発者は利用者の不適切な利用に起因する責任を負いません
